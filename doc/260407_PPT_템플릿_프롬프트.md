교회 QT(Quiet Time) 묵상용 PowerPoint 템플릿을 만들어 주세요.

목적:
- 설교/묵상 내용을 정리하는 한국어 QT 문서 템플릿
- 이후 Python 프로그램이 JSON 데이터를 넣어 자동으로 PPTX를 생성할 예정
- 따라서 구조가 명확하고 편집이 쉬워야 함

디자인 방향:
- 깔끔하고 단정한 교회/묵상 느낌
- 따뜻하고 차분한 분위기
- 밝은 배경과 충분한 여백
- 한국어 문단 가독성이 높은 텍스트 중심 디자인
- 과도한 장식, 복잡한 도형, 기업 보고서 느낌은 피할 것

슬라이드 구성:
1. 표지 슬라이드: 제목, 본문 성구, 찬송
2. 말씀의 창 슬라이드: 문단 2~4개 정도가 들어가는 구조
3. 오늘의 메시지 슬라이드: 상단 제목은 "오늘의 메시지", 본문 첫 줄은 "1. 메시지 제목" 형식, 그 아래 일반 문단 2~4개
4. 오늘의 메시지 슬라이드: 같은 구조로 반복 가능
5. 삶의 적용 슬라이드: 번호형 문장 3개 정도
6. 오늘의 기도 슬라이드: 불릿이 아닌 일반 문단 2~3개

중요:
- 글머리표보다 일반 문단 중심
- 제목 영역과 본문 영역을 명확히 분리
- PowerPoint에서 수정하기 쉬운 템플릿
